Developer - POJO класс, который содержит данные о разработчике
Класс DeveloperDAO - содержит методы для работы с текстовым файлом //файл
Класс DeveloperView - который содержит логику работы с консолью //основа
Класс DeveloperController - экземпляр которого создаётся в DeveloperView
для обработки запросов из консоли и содержит DeveloperDAO.
Программа должна прекращать работу только по указанию пользователя
Выход из программы в результате исключения недопустим.

1. Создание разработчика
2. Редактирование
3. Получение данных (чтение)
4. Удаление

view <- controller(запроси из консоли)
из консоли основи(поля null) визиваєм controler(поля null) (запуск файла (поля: имя, возрост, посада, стаж)) і записуєм в developer
